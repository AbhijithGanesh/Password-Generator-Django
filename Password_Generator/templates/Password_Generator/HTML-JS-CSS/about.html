About Team UnHackables

The idea behind this project is to generate unhackable passwords/passphrases intended to be used by Banking and other applications which requires a very-secure application

This project will generate a 128-bit passcode with various bits and characters

The passwords were tested in various password cracking application(s) and were nearly impossible to crack

Various concepts such as Python Object Oriented Programming,Random Module, Modules and Libraries, Django and usage of Mersene Prime numbers to generate a password was implemented
